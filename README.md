# Bazel 系统学习项目

从零到精通的 Bazel 学习项目集合，包含 22 个实战项目。

## 项目列表 (22/22 已完成)

**基础入门 (1-6)**: Hello World (C++/Python), 多文件程序, 模块依赖, 计算器, 多语言构建

**依赖管理 (7-8)**: 跨 package 依赖, 可见性控制

**测试 (9-10)**: 测试套件, 工具链版本管理

**配置与生成 (11-14)**: 条件编译, debug/release 模式, genrule, 模板生成

**自定义规则 (15-16)**: 宏 vs 规则, Provider 机制

**高级特性 (17-22)**: CI/CD 集成, 远程缓存, 打包规则, 构建变体, 平台工具链, 代码生成

## 开始学习

```bash
cd project01-hello-cpp && bazel build //:main
```
